# Keyboard Reference

All panel shortcuts show default key bindings. Every
action can be remapped via settings.
See [Settings and Themes](Settings.md).

## Global Keys

  `Ctrl+O`       Toggle file manager panel on/off
  `Alt+Enter`    Detach to fullscreen / reattach

## Navigation

  `Up` / `Down`           Move cursor through file list
  `Left` / `Right`        Move between columns; scroll
  `PgUp` / `PgDn`         Scroll by one full page
  `Home`                  Jump to first entry
  `End`                   Jump to last entry
  `Tab`                   Switch between panes

## Actions

  `Enter`    Open directory / archive / file / execute cmd
  `F1`       Help -- open this help system
  `F3`       View -- highlight in VS Code Explorer
  `F4`       Edit -- open file in VS Code editor
  `F5`       Copy files to other pane
  `F6`       Move or rename files
  `F7`       Make directory
  `F8`       Delete (Trash on Windows/macOS, permanent
             on Linux/FreeBSD)
  `Shift+F8` Permanently delete (bypass Trash)
  `F9`       Open top menu bar
  `F10`      Quit VSCommander

## Panel Controls

  `Ctrl+H`              Toggle dotfile visibility
  `Ctrl+P`              Half-panel mode (show terminal)
  `Ctrl+Q`              Quick View (preview in split)
  `Ctrl+R`              Re-read active pane directory
  `Ctrl+U`              Swap left and right panes
  `Ctrl+1` / `2` / `3`  Set column count (1, 2, or 3)
  `Ctrl+Left` / `Right` Resize pane border

## Selection

  `Insert`       Toggle selection, move cursor down
  `Numpad *`     Invert selection (toggle all)
  `Numpad +`     Select all files
  `Numpad -`     Deselect all files
  `Shift+Up`     Select range upward
  `Shift+Down`   Select range downward

See [File Selection](Selection.md) for details.

## Quick Search

  `Alt+letter`   Open quick search, jump to match

See [Quick Search](QuickSearch.md) for details.

## Drive Popup

  `Alt+F1`       Change drive popup (left pane)
  `Alt+F2`       Change drive popup (right pane)

See [Change Drive](DriveDlg.md) for details.

## Sorting Shortcuts

  `Ctrl+F3`      Sort by name
  `Ctrl+F4`      Sort by extension
  `Ctrl+F5`      Sort by modification date
  `Ctrl+F6`      Sort by size
  `Ctrl+F7`      Unsorted (filesystem order)
  `Ctrl+F8`      Sort by creation time
  `Ctrl+F9`      Sort by access time
  `Ctrl+F10`     Sort by description
  `Ctrl+F11`     Sort by owner
  `Ctrl+F12`     Open Sort Modes popup
  `Shift+F11`    Toggle sort groups
  `Shift+F12`    Show selected first

See [Menus and Sorting](Menus.md) for details.

## Command Line

  Printable keys   Type into command line
  `Backspace`      Delete character
  `Ctrl+Enter`     Insert filename into command line
  `Ctrl+[`         Insert left pane path
  `Ctrl+]`         Insert right pane path

See [Command Line](CmdLine.md) for details.
