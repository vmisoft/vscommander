# Mouse Interactions

Mouse tracking is enabled while the panel is visible and
disabled when you switch to the shell, so normal text
selection works outside the panel.

## File Area

  Left-click         Move cursor to entry; clicking the
                     inactive pane switches focus to it
  Double-click       Open directory or file (same as Enter)
  Click and drag     Cursor follows the mouse

## Function Key Bar

  Left-click         Triggers the corresponding F-key

## Popups

  Left-click button  Activates the clicked button
  Left-click control Focuses the clicked field
  Drag popup body    Moves the popup around the screen
  Click outside      Closes the popup
  Scroll wheel       Scrolls popup lists

## General

  Scroll wheel       Move cursor up/down by 3 entries
  Middle-click       Same as Enter (open dir/file/exec)
