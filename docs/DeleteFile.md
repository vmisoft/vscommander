# Delete Files

## Delete (F8)

Press `F8` to delete the selected file or directory.

- Windows and macOS: Moves to Recycle Bin / Trash by
  default, with option to permanently delete
- Linux and FreeBSD: Permanently deletes (no trash)

A confirmation dialog shows the file name and available
options. A progress dialog appears during deletion of
large directories.

## Permanent Delete (Shift+F8)

Press `Shift+F8` to permanently delete, bypassing the
Recycle Bin / Trash on all platforms.

## Cancellation

Press `Esc` during deletion to cancel the operation.
Files already deleted are not restored.
